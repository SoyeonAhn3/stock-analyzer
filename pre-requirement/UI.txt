1. 디자인 방향
================================================================================

[테마]
  라이트 모드 기본.
  컬러: 파랑 계열(Primary) + 회색 계열(Neutral) + 흰색(Background)
  톤: 깔끔하고 신뢰감 있는 금융 대시보드.
  과하게 화려하지 않되, 데이터가 한눈에 들어오는 구조.

[컬러 시스템]

  Primary (파랑):
    50:  #eff6ff   — 카드 배경 (활성 상태)
    100: #dbeafe   — 보더 (활성)
    200: #bfdbfe   — 보더 (호버)
    300: #93c5fd   — 보조 장식
    400: #60a5fa   — 아이콘, 차트 라인
    500: #3b82f6   — 버튼, 강조
    600: #2563eb   — 버튼 (hover), 활성 모드
    700: #1d4ed8   — 텍스트 강조, 로고

  Neutral (회색):
    50:  #f8fafc   — 페이지 배경
    100: #f1f5f9   — 카드 내부 구분선, 비활성 배경
    200: #e2e8f0   — 보더 기본
    300: #cbd5e1   — 비활성 텍스트 아이콘
    400: #94a3b8   — 보조 텍스트 (라벨, 힌트)
    500: #64748b   — 보조 텍스트 (설명)
    600: #475569   — 본문 텍스트
    700: #334155   — 제목 텍스트
    800: #1e293b   — 강한 제목
    900: #0f172a   — 최상위 제목

  Semantic (의미 색상):
    상승/긍정:  #16a34a (green-600)
    하락/부정:  #dc2626 (red-600)
    주의/중립:  #d97706 (amber-600)
    상승 배경:  #f0fdf4 (green-50)
    하락 배경:  #fef2f2 (red-50)
    주의 배경:  #fffbeb (amber-50)

[타이포그래피]

  폰트: DM Sans (1순위), Segoe UI (2순위), sans-serif (폴백)
  이유: DM Sans는 숫자 가독성이 좋고 금융 대시보드에 적합.

  크기 체계:
    28~30px  — 주가 (메인 가격 표시)
    22px     — 페이지 제목
    16px     — 섹션 제목, 큰 KPI 값
    14px     — 카드 제목, 테이블 데이터
    13px     — 본문 텍스트
    12px     — 보조 텍스트, 차트 라벨
    11px     — 태그, 뱃지, 힌트 텍스트
    10px     — 면책 조항, 데이터 지연 표시

  무게:
    700  — 주가, 티커명, 페이지 제목
    600  — 카드 제목, KPI 값, 등락률
    500  — 섹션 소제목, 태그
    400  — 본문, 설명 텍스트

[간격 체계]

  카드 내부 패딩: 16px
  카드 간 간격: 12px
  섹션 간 간격: 16~20px
  테이블 셀 패딩: 8px
  인라인 요소 간격: 8~10px
