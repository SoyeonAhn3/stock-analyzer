### TC-004 [정상] 👤 수동 — Quick Look 전체 데이터 표시 확인

**사전조건**: Quick Look 화면 진입 (`/quick-look/AAPL`)
**실행 단계**:
1. PriceHeader 영역 확인 (가격, 등락, 52W Range 바)
2. KPI 카드 4개 확인 (Market Cap, P/E, EPS, Forward P/E)
3. 캔들스틱 차트 확인 (OHLC + Volume + MA50 + MA200)
4. Technical Cards 확인 (RSI, MACD, Bollinger Bands)
5. AI Recommendation 카드 확인
**입력값**: 티커 `AAPL`
**예상 결과**:
- PriceHeader: 현재 가격(큰 폰트), 등락액/등락률, 52주 최저~최고 레인지 바 + 현재 위치 마커
- KPI 카드: 4개 카드에 각각 수치와 라벨 표시, 숫자는 JetBrains Mono 폰트
- 차트: 캔들(녹색=양봉, 빨간=음봉), 하단 볼륨 히스토그램, 파란 MA(50)선, 주황 MA(200)선
- Tech Cards: RSI 값 + bullish/bearish/neutral 시그널 뱃지, MACD 히스토그램 설명, Bollinger 밴드 수치(upper/middle/lower)
- AI Recommendation: "AI Deep Analysis" 버튼 표시
