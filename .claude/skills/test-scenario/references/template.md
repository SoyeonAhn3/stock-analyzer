# 시나리오 파일 템플릿

## 파일 경로 규칙

```
test-scenarios/YYYYMMDD_기능명.md
```

- 날짜: 오늘 날짜 (currentDate 기준, YYYYMMDD 형식)
- 기능명: 사용자가 설명한 기능을 2~4단어로 요약, 한글 가능
- 예: `test-scenarios/20260313_LocalStorage저장.md`
- 예: `test-scenarios/20260313_Sidebar뷰전환.md`

---
