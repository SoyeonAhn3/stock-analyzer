# 면접 준비 파일 템플릿

## 파일 경로 규칙

```
interview/YYYYMMDD_프로젝트명_면접준비.md
```

- 날짜: 오늘 날짜 (currentDate 기준, YYYYMMDD 형식)
- 프로젝트명: 분석한 프로젝트 이름을 2~4단어로 요약, 한글 가능
- 예: `interview/20260320_ProcessFlow생성기_면접준비.md`
- 예: `interview/20260320_SkillPackage_면접준비.md`

```markdown
## 면접 전 체크리스트

- [ ] 프로젝트 구조를 직접 설명할 수 있는가
- [ ] 기술 선택 이유를 논리적으로 설명할 수 있는가
- [ ] 본인이 기여한 부분을 구체적으로 말할 수 있는가
- [ ] JD 요구사항과 프로젝트 경험을 연결할 수 있는가
- [ ] 프로젝트의 한계점과 개선 방향을 알고 있는가
- [ ] 꼬리 질문에 대비했는가

```
